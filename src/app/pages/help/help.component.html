eServices Assistant is designed to assist the internal user in performing their work with rapid ease.



Guide:

  • 1.   A user can login using an eServices login ID.
  • 2.  A user can perform multiple tasks depending on the rights available to the user. e.g. Some users can only Transfer a process, some can activate or delete a company user and some can perform all tasks.



Below is how the screen will look like for the users who have rights to perform all tasks.




Process Transfer:


User Process ID:

Process transfer requires a “User Process ID” and “New Owner” name to transfer a process. A “User Process ID” is a unique identification number of the process which can be extracted from the process name listed in web client (Internal Application).


Consider this example.



In the above, the “User Process ID” is 1426522.


New Owner:

New Owner can be selected from the list available.



Enter “User Process ID”, select the “New Owner” and then press the “Transfer Process” button to transfer the process to the new owner.


Activate Company User:

Enter the company user name and press “Activate User “ to activate.


Delete Company User:

Enter the company user name and press “Delete User “ to delete.
