Help





eServices Assistant is designed to assist the internal user in performing their work with rapid ease.



Account Creation:

An administrator can create a new account for a user in order to grant the access to this application.




Login:

A user can login using an eServices login ID that the administrator provided to him at the time of account creation.

Reports:

A user can perform multiple tasks depending on the rights available to the user. e.g., Some users can only View Company Records, some can Search Bank Transaction Log and some can view CTC Filing Status Report, etc.

There are multiple reports available in the application. For example, take a look at the CTC Filing Status Report




User Profile:

A user can view his/her profile as well as upload his/her profile picture. However, he/she has to initiate a request to an administrator in order to change his profile information.



Request Log:

An administrator has the right to update a user's profile information.
